Scale the data. This is recommended in most cases so that one highly variable feature doesn't dominate the analyses. This is equivalent to running PCA on the correlation rather than the covariance matrix

In [4]:
x_scaled = StandardScaler().fit_transform(x)

Run dimension reduction methods (PCA, t-SNE and UMAP)
